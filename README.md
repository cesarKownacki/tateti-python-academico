# 🕹️ Ta-Te-Ti (Tic-Tac-Toe) - Proyecto Académico 2024

Este es un proyecto nostálgico desarrollado en **2024** para la materia **"Algoritmos y Programación"**. Representa mis primeros pasos sólidos en la lógica de programación y el manejo de estructuras de datos bidimensionales con Python.

## 📝 Historia del Proyecto
El script nació como un desafío para aplicar conceptos de matrices y validación de errores. Lo más valioso de este código (y lo que conservo con orgullo) es la sección de "Errores a Corregir" documentada en el código fuente, que refleja el proceso de debugging que realicé en su momento:
* Validación de coordenadas fuera de rango.
* Prevención de sobrescritura de jugadas.
* Detección temprana de victorias y empates.

## 🛠️ Tecnologías Utilizadas
* **Python 3**
* **NumPy**: Utilizado para la generación y gestión de la matriz de juego.
